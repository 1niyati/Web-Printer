
**PRINTER TECH E-COMMERCE WEBSITE DEVELOPMENT**

📌 **About This Project**
The Printer Tech e-commerce website was a collaborative group project that involved building a fully operational online platform for printer sales using HTML, CSS, and JavaScript. The site comprised 7 interactive pages incorporating responsive layouts, a shopping cart system, user login functionality, and comprehensive testing across all features.

**Technologies Used**
Frontend: HTML5, CSS3, JavaScript — Design Tools: Balsamiq Wireframes, Visual Studio Code — Features: Animated content sliders, input validation, popup notifications, dynamic navigation menus

**Key Features Developed**

**Home Page:** Engaging hero banner with call-to-action buttons, highlighted product listings, and promotional deals with discount displays.

**Product Catalogue:** Full printer inventory with in-depth specifications, clear pricing structures, and seamless add-to-cart functionality.

**Blog Section:** Informative articles exploring the benefits, limitations, and real-world uses of various printing technologies.

**User Authentication:** Account login interface featuring input validation, error handling, and real-time user feedback.

**Research Documentation:** Dedicated section presenting reference materials alongside final design decisions to demonstrate development rationale.

**Project Outcomes**
Delivered a polished, industry-standard e-commerce platform showcasing strong practical knowledge of web development and information systems. Built valuable experience in team-based software development, task coordination, and current web technologies. Produced a well-structured, expandable codebase with a focus on intuitive user experience, adaptive design, and engaging interactivity.
